Java-swing-Contect-Form
Java Swing Contact Form with input validation, radio buttons, combo box, and message text-area.

This project is a GUI-based Contact Form developed using Java Swing. It allows users to enter personal details such as name, phone number, city, gender, and a message.
The application demonstrates the use of various Swing components along with input validation and event handling.

Features:
- User-friendly graphical interface
- Input validation for:
  - Name (must start with a capital letter)
  - Phone number (numeric only)
- Gender selection using radio buttons
- City selection using JComboBox
- Message input using JTextArea with scroll functionality
- Reset button to clear all fields
- Cancel button to close the application
- Error handling using dialog boxes

Technologies Used:
- Java
- Java Swing
- AWT Event Handling
- SpringLayout (for GUI layout management)

How It Works:
1. User enters all required details
2. Clicks the "Submit" button
3. System validates the inputs:
   - Displays error messages if invalid
   - Shows success message if valid
4. Data is printed in the console upon successful submission

Learning Outcomes:
- Implementation of Java Swing components
- Handling user input and validation
- Event-driven programming using ActionListener
- Layout management using SpringLayout
